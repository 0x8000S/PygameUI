# PygameUI

一个轻量级、面向对象的 Pygame GUI 框架。![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

### 特性

- 🎯 **极简 API** — 几行代码创建交互界面
- 🧩 **组件化设计** — Widget 基类易于扩展
- ⚡ **自动管理** — 全局管理器处理事件、更新、渲染
- 🎨 **主题系统** — 颜色、字体统一配置
- 📐 **自动布局** — 支持根据内容自动计算尺寸

本模块目标为为pygame添加ui的支持，方便游戏内UI的创建
